# Hyperskill Cafe

Hyperskill Cafe is a semi-dynamic, fully responsvie website based on bootstrap, created as part of the JetBrains Academy Frontend Developer course - https://hyperskill.org/projects/210.
<br><br>
Tests are provided by JetBrains.
